消防智能场景预案管理系统 - 使用说明

一、首次使用前准备

1. 请确认电脑已安装：
   - Python
   - Node.js
   - Microsoft Word

2. 如果项目是从其他电脑复制过来的，请不要随意修改目录结构。


二、启动系统

最简单的方式：

1. 双击 start_all.bat
2. 等待弹出两个窗口
3. 浏览器打开 http://localhost:5173

说明：

- 一个窗口是后端服务
- 一个窗口是前端页面服务
- 使用过程中请不要关闭这两个窗口


三、如果只想单独启动

1. 双击 start_backend.bat
   - 启动后端服务

2. 双击 start_frontend.bat
   - 启动前端服务


四、同步预案

如果新增了预案文件：

1. 把新的 .doc 文件放到“预案”目录下
2. 打开系统网页
3. 点击右上角“同步预案目录”

系统会自动扫描并导入新的预案。


五、预案分类

预案文件可以放在以下任意层级：

- 预案根目录
- 支队目录
- 支队/大队目录
- 支队/大队/中队目录

网页顶部的预案下拉选择会自动按支队 / 大队 / 中队显示分类。


六、关闭系统

关闭系统时，直接关闭两个黑色命令窗口即可。


七、常见问题

1. 双击启动脚本没有反应
   - 请确认电脑已安装 Python 和 Node.js

2. 页面打不开
   - 请确认两个启动窗口都没有报错
   - 再访问 http://localhost:5173

3. 同步失败
   - 请确认电脑已安装 Microsoft Word
   - 请确认预案文件不是正在被其他程序占用


八、技术支持

如果系统无法正常启动或页面显示异常，请把报错信息截图后反馈给维护人员。
